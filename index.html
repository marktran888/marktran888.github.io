General Assembly Projects

Project 4 - Ingredients Scanner

https://ingredients-check.herokuapp.com
https://github.com/marktran888/WDI_LDN_PROJECT4

Technologies used: MERN stack, AWS Rekognition API, HTML5, CSS3, AJAX, Bulma, Chai, Mocha, Yarn

The user can create and maintain a list of food ingredients they are allergic to and then upload photos of ingredients list to be scanned.
I built out the full-stack app, using Express API to serve user data from a Mongo database and used AWS Rekognition API to read and process the images. I built out the front-end with React and automated tests to test both back and front end components.

Project 3 - Make My Day

https://make-my-day-project.herokuapp.com
https://github.com/marktran888/WDI_LDN_PROJECT3

Technologies used: MEAN stack, Google Maps/Places API, HTML, CSS, Bulma, AJAX, Bulma, Chai, Mocha, Yarn

The user chooses a holiday destination and suggestions places to visit will appear on a map. The user then selects what they would like to visit and a route is planned out. This was a group project with 2 other people. We used Agile/Scrum processes to plan and organise the work and we attempted to split the work so that everyone could help with parts of designing, creating API RESTful routes, creating the front end interface and the Google API.
The full-stack app uses MongoDB, Node.js and Express for the server-side API and includes all the RESTful states and Angular to build out the front-end. We also wrote automated tests using Mocha and Chai.
